# Stellar Coordinate Explorer
A Python-based tool for transforming and visualizing stellar coordinates across different astronomical reference frames using the Astropy library.

## Overview
Astronomical data is recorded in multiple coordinate systems (e.g., ICRS, Galactic, Ecliptic) and converting between them is essential for analysis and interpretation.

This project explores how to:
- Transform stellar coordinates between reference frames
- Work with real astronomical data
- Visualize positions of stars in different coordinate systems

## Objectives
- Implement coordinate transformations (ICRS, Galactic, AltAz)
- Load and process real star catalog data
- Generate visualizations of stellar positions
- Build a reusable workflow for astronomical data analysis

## Tools and Technologies
- Python
- Astropy
- Numpy, Pandas
- Matplotlib
- Streamlit

## Status
Project initialized - core coordinate transformation functions in development.

## Planned Features

## Why This Project Matters
This project demonstrates practical applications of computational astronomy, specifically how coordinate systems are used to interpret and analyze observational data.
